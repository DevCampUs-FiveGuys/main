## 팀원 역할 📦

### ⭐️ 유상곤
- [포트폴리오] <br>
<b style="color :#00C300">페이지 : 포트폴리오(메인,작성,수정,세부) 페이지</b> <br>
포트폴리오 작성/수정/삭제 기능, 포트폴리오 메인페이지에서 다른 사람이 작성한 포트폴리오 확인 가능하고 클릭 시 해당 포트폴리오의 세부 페이지로 이동


### ⭐️ 허승필
- [학생 마이페이지] <br>
<b style="color :#00C300">페이지 : 출석 현황, 찜목록, 게시글, 정보 수정</b> <br>
수강생 입/퇴실 또는 휴가 신청 기능과 본인이 찜한 포트폴리오나 작성한 포트폴리오 확인 가능, 회원가입시 작성한 정보들 불러오고 전화번호와 프로필 사진 수정 가능

### ⭐️ 정진욱
- [로그인/회원가입] <br>
<b style="color :#00C300">페이지 : 로그인, 회원가입 페이지</b> <br>
Spring Security를 이용하여 로그인/회원가입 페이지 구성, 회원 가입시 비밀번호 암호화 상태로 DB에 저장<br>

- [메인 페이지] <br>
<b style="color :#00C300">페이지 : 메인 페이지</b> <br>
메인 페이지 구성과 전체적인 템플릿,레이아웃 작성

- [깃/배포] <br>
깃허브 및 배포 사이트 개설

### ⭐️ 민경진
- [문의 게시판] <br>
<b style="color :#00C300">페이지 : 메인 페이지, 수정 페이지, 추가 페이지</b> <br>
관리자 권한을 가지고 있을 시 문의 게시글 추가/수정/삭제 버튼 생성, 다른 일반 계정들은 문의 게시글 열람 권한만 가능<br>

- [강사 마이페이지] <br>
<b style="color :#00C300">페이지 : 출석 현황, 출석 명단, 학생 승인, 정보 수정</b> <br>
총 출석 인원, 휴가 사용 인원, 출석한 인원의 입,퇴실 정보, 게스트 계정을 학원으로 권한 변경 가능

### ⭐️ 박민지
- [리뷰 페이지] <br>
<b style="color :#00C300">페이지 : 리뷰 페이지</b> <br>
과정명과 기수를 선택하여 후기 검색 별점과 함께 후기 작성, 평균 평점, 점수별 비율, 성별 비율 <br>

- [관리자 마이페이지] <br>
<b style="color :#00C300">페이지 : 출석 현황, 권한 변경, 정보 수정, 훈련 정보</b> <br>
관리자 권한을 가지고 있을시 관리자 마이페이지로 이동, 관리자는 다른 계정의 권한 변경이 가능하고 출석현황에서 휴가 또는 병가 승인이 가능하며 여러 부트캠프의 훈련 과정 추가 가능<br>
